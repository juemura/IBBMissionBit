Course Overview and Objectives

Course Description

This is a mobile game design course in which students will learn to design and implement a Java game application for Android devices using libGDX, a popular game framework. There will be several guest speakers from the game industry who will talk about game production, success and failure stories, and also technical talks on various game components. You'll get to work with and learn from mentors who are professional software engineers, and get feedback from teams at game development studios. By the end of the course you'll have a playable mobile game, ready for release in the app store.

Topics will include Java programming techniques, the Java Android software development kit (SDK), LibGDX, principles of game design, and version control using git and Github.

Learning Goals

  • Technical skills:
    • Prototype and implement an android game
    • Describe the main features of Java
    • Write mobile games for Android powered by libGDX
    • Use the Android development toolchain to write and debug applications
    • Independently learn about and utilize external libraries and APIs
    • Use git and github for version control and code sharing (team / open source workflow - pull, push, add, commit, PRs)
  • Other goals:
    • Students will work effectively on teams and review each other's code.
    • Students will feel like they belong in tech
